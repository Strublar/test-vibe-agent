# Hello World Application

Une application JavaScript simple qui affiche "Hello World" dans la console.

## 📋 Prérequis

- Node.js >= 14.0.0
- npm (inclus avec Node.js)

## 🚀 Installation

1. Clonez ou téléchargez ce projet
2. Naviguez vers le dossier du projet :
   ```bash
   cd test-vibe-agent
   ```

## 🎯 Utilisation

### Méthode 1 : Avec npm
```bash
npm start
```

### Méthode 2 : Directement avec Node.js
```bash
node hello.js
```

### Méthode 3 : Comme exécutable (Linux/macOS)
```bash
chmod +x hello.js
./hello.js
```
